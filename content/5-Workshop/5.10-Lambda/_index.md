#### Overview

Create and test an AWS Lambda function for your backend logic.

### Steps

1) In **Lambda Console** → **Create function** → **Author from scratch**  
   ![Lambda](/images/5-Workshop/5.10-Lambda-img/lambda1.jpg)

2) Name the function, choose **Runtime** (e.g., Node.js/Python), keep default permissions or select an execution role  
   ![Lambda](/images/5-Workshop/5.10-Lambda-img/lambda2.jpg)

3) Add/modify the **handler code** in the editor  
   ![Lambda](/images/5-Workshop/5.10-Lambda-img/lambda3.jpg)

4) Click **Deploy** to publish the latest code  
   ![Lambda](/images/5-Workshop/5.10-Lambda-img/lambda4.jpg)

5) Create a **Test event** and run **Test** to verify output
